Where Do Illegal Guns Come From?

Each year approximately 250,000 firearms are reported stolen in the United States. Based on the ATF Tracing Program, approximately half that number are recovered each year, meaning there is an ever-growing stockpile of illegal arms that numbers in the millions.

Using ATF Tracing Data, we have tracked where these illegal guns have originated. Some states contribute more to the illegal arms stock pile than others, and there appears to be a correlation between the number of illegal guns that originate from a state and local legislation supporting gun rights.

Using a scoring model created by Guns & Ammo Magazine, we mapped the current status of gun rights against the number of illegal guns originating from that state
